# School-Projects

## Building
* Requires CMake 3.0+
* C++11 compiler
* OpenGL 4.0+

1. Run `CMake ..` from *lab-env/build* folder
2. Executable is added to *lab-env/bin* (Set *lab-env/bin* as working directory if running from a debugger)
