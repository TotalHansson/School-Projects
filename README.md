# School-Projects

## Building
*Requires CMake 3.0+
*C++11 compiler
*OpenGL 4.0+

1. Create a build folder in *lab-env*
2. Run `CMake ..` from the build folder
3. Executable is added to *lab-env/bin* (Set *lab-env/bin* as working directory if running from a debugger)
